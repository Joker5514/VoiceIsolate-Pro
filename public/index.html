Spectral Subtraction

64-band frequency-domain noise removal using over-subtraction with a spectral floor to eliminate musical noise artifacts.

|X̂(ω)| = max(|X(ω)| - α·E[|N(ω)|], β·|X(ω)|)

Wiener Filtering

Adaptive MMSE Wiener gain applied per frequency bin — strongest suppression in low-SNR regions for natural-sounding output.

G(t) = √( max(P_signal - P_noise, 0) / max(P_signal, ε) )

Silero VAD + ONNX Runtime ML

ONNX Runtime Web powers real-time Voice Activity Detection in a dedicated Web Worker — classifying speech frames at sub-millisecond latency with zero network calls.

P(voice | frame) → gate mask @ 20 ms chunks · ONNX WebGL backend

Dynamics & Compression

Envelope-follower compressor with per-sample attack/release constants keeps dialogue intelligible across wide dynamic range content.

a = exp(-1 / (SR · t_attack)) · GR = -max((L - T)·(1 - 1/R), 0)

LUFS Normalization

Output is normalized to broadcast targets — -14 LUFS for streaming, -16 LUFS for podcasts — using ITU-R BS.1770-4 K-weighted measurement.

gain = 10^( (L_target - L_measured) / 20 )
