## 2026-03-01 - Native Range Slider Accessibility\n**Learning:** The native HTML `<input type="range">` lacks clear visual focus indicators by default in many browsers and CSS resets. Adding `focus-visible:ring` classes ensures keyboard users can track their position when tabbing through multiple adjacent sliders.\n**Action:** Always apply explicit `focus-visible` utility classes to range inputs, especially in dense control panels, alongside proper `id`/`htmlFor` label associations.
